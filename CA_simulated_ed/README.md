# California Simulated ED Analysis

## Purpose

This folder contains code for simulating HRI ED daily counts across California ZCTAs and validating the ED modeling pipeline before application to observed data. The simulation evaluates whether the approach can recover cluster-level differences in HRI ED burden introduced through assigned risk differences.

---

## Files

| File | Description |
|---|---|
| `generating_data.R` | Generates simulated ZCTA-level HRI ED daily counts |
| `analysis.R` | Fits Poisson and negative binomial GAMs |

| `cluster_multiplier.csv` | Contains assigned simulated cluster risk multipliers and vulnerability scores |

## Simulation Approach

- Cluster-specific vulnerability scores were calculated from SES characteristics and used to assign simulated HRI ED risk multipliers, with higher vulnerability clusters receiving larger multipliers.
- Individual ED visits were simulated at the event level, with each visit assigned a heat-related ICD code (dehydration, kidney failure, or heat illness/stroke) based on fixed proportions, then aggregated to ZCTA-day counts.
- Daily expected visit counts were built from:
  - ZCTA population size
  - assigned cluster-level risk multipliers
  - a seasonal curve (summer peak) rescaled each year to match real, age-adjusted HRI ED rates observed for that year (2008–2018)
  - ZIP-level random variation to reflect within-cluster heterogeneity
  - ZIP-day-level random variation to introduce overdispersion
---

## Modeling Approach

The model was designed to mirror the temporal structure of CalHeatScore's spatiotemporal regression framework. Models estimated cluster-level differences in HRI ED burden while accounting for:

- Population size using an offset term to model ED visit rates rather than raw counts
- Year effects
- Day-of-week effects
- Day of year using a GAM smooth term to capture nonlinear seasonal variation in HRI ED risk

Models evaluated:

1. Poisson GAM
2. Negative binomial GAM

The negative binomial model was selected based on lower AIC and its ability to account for overdispersion in ED visit counts. A zero-inflation diagnostic indicated that the negative binomial model adequately captured the proportion of zero-count days; therefore, a zero-inflated model was not used.

---

## Validation

Estimated cluster IRRs were compared with the assigned simulated IRRs to evaluate whether the model recovered the differences introduced during simulation. This validation assesses the performance of the analysis pipeline and does not represent findings from observed ED data.
